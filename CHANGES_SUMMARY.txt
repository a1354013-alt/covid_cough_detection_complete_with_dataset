SEMANTIC ALIGNMENT
================================================================================

Health Check Endpoints:
  /api/healthz (liveness)  → Node.js process alive
  /api/readyz (readiness)  → Node.js + Python + Model ready
  /api/health (backward)   → Mirrors /readyz

Error Differentiation:
  Python running, model not ready  → status: "not_ready", python_backend: "started"
  Python unreachable               → status: "not_ready", python_backend: "unreachable"
  Everything ready                 → status: "ready", python_backend: "ok"
